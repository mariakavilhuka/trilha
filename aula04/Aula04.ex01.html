TECNOLOGIAS

HTML 5

Definição: uma linguagem de marcação de hipertexto, responsável pelo conteúdo em exibição no site, por exemplo: TEXTOS, TABELAS, IMAGENS e VÍDEOS.

CSS 3

Ferramenta responsável pela estilização do conteúdo, exemplo: cor, tamanho, fonte, sombra e posicionamento.

JAVA SCRIPT

Responsável pela interação com o conteúdo, como: menus interativos, validação, pop-ups e animações.

Pelo URL da página, é enviada uma requisição para um servidor DNS, que busca o IP do site, retornando para o cliente, o cliente envia uma requisição ao servidor do site, que disponibiliza os arquivos HTML, CSS e JAVA SCRIPT, cada arquivo desempenha um papel, formando o site.

PÁGINA WEB e WEBSITE

PÁGINA WEB é um arquivo que possibilita que o usuário insira elementos, como: textos, imagens, tabelas, vídeos, etc, que tenham ligações de hipertexto que permitam a transição de uma página para outra

WEBSITE é uma coleção de páginas web, armazenadas em pastas vinculadas a um servidor

O sistema funciona em lado do cliente e lado do servidor, onde o cliente realiza uma requisição e o servidor processa-a e retorna ao cliente. Para isso, é necessário que sejam feitas por profissionais da área.

HTML

Linguagem de Marcação de Hipertexto

Criada na década de 90, por Tim Berners-Lee, possibilitou a publicação de conteúdo em hipertexto, podendo ser vinculadas por meio de hyperlinks

TAGS

Definem a marcação do conteúdo, cada tag tem funcionalidades distintas. Podem possuir atributos, como parâmetros para o navegador.Sem o uso das mesmas é exibido um texto plano.

O início do programa se dá com a tag HTML, dentro desta HEAD, que define o cabeçalho, dentro da mesma TITLE, que define uma barra de título para o navegador e dentro desta BODY, onde fica o conteúdo.

Para adicionar comentários, usa-se o parênteses angulado, sinal de exclamação e dois traços, depois do comentário dois traços e fecha-se o parênteses angulado

DOCTYPE

É a TAG que declara o tipo do documento.

Dentro da tag HEAD, indroduz-se as tags meta, que definem, por exemplo, o tipo de tela utilizada

MARCAÇÃO DE TÍTULOS

Utiliza-se as tags Heading (h), de 1 até 6

MARCAÇÃO DE PARÁGRAFOS

Utiliza-se as tags de Parágrafo (p), para iniciar e finalizar parágrafos

MARCAÇÃO DE QUEBRA DE LINHA

Utiliza-se as tags de Quebra de linha (br)

MARCAÇÃO PARA FORMATAÇÃO DE TEXTO

STRONG, para texto destacado

I, para texto em itálico

EM, para texto em ênfase

BIG, para aumentar o texto

FONT COLOR, para inserir cores no texto
