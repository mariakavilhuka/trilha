Branches e Hospedagem

Branches

Branches são ramificações de um projeto de software ou documento e tem como objetivo o desenvolvimento paralelo de:

  • Novas funcionalidades
  • Correção de problemas
  • Alteração de código

Branches

Branch Master ou Main:
Braço principal de um projeto ou repositório.
Branch Secundária:
Braço secundário de um projeto em repositório.

Hospedagem

O Github pages oferece hospedagem gratuita de sites. Basta criar uma conta e um repositório público e a URL será: https://nomeusuario.github.io/repositorio
