Hyperlinks

Definição: é uma referência dentro do documento em hipertexto para outras partes do mesmo documento, outros documentos, arquivos ou endereço eletrônico. Os links podem ser:

  • remotos
  • internos
  • locais
  • com endereço de arquivo

Marcação

A marcação de hiperlinks acontece utilizando a tag < a > e a propriedade href que faz a referência para o conteúdo desejado.

Hiperlinks externos ou remotos

São links que referenciam sites ou endereços da web. Na propriedade href utilizamos a URL (Uniform Resource Locator) para a referência que desejamos.

Sintaxe: < a href = " U R L " > Conteúdo Referenciado < / a >

Hiperlinks internos

São links que apontam a um lugar diferente dentro da mesma página e possibilita acessar diferentes parágrafos ou seções quando o documento é extenso.

Sintaxe: < a href = " referência " > Conteúdo Referenciado < / a >

< a name = " referência " > < / a >

Hiperlinks locais

São links que apontam para outras páginas dentro de um mesmo site.

Sintaxe: < a href = " arquivo . html " > Conteúdo Referenciado < / a >

Hiperlinks em imagens

É possível realizar hiperlinks imagens envolvendo a tag de imagem com a tag de hiperlink

Sintaxe: < a href = " referência " > < img src = " imagem . jpg " > < / a >

Hiperlinks para downloads de arquivos

São links que apontam para arquivos e possibilitam realizar download desses arquivos

Sintaxe: < a href = " arquivo . extensão " download = " arquivos . extensão" type = " tipo "> Conteúdo < / a >
